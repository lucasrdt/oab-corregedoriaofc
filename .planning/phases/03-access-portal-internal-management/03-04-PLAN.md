Criar `src/pages/portal/CasoEditor.tsx`. Este componente é usado em três contextos:
- `/portal/presidente/casos/new` — criação (presidente)
- `/portal/presidente/casos/:casoId` — edição (presidente) ou visualização (user via admin)
- O mesmo componente serve para admin e presidente (RLS do banco resolve o acesso)

**Estrutura do componente:**

1. Usar `useParams()` para extrair `casoId` da URL
2. Usar `useAuth()` para obter `role`, `subsectionId`
3. `readonly` mode: `role === 'user'` — ocultar botões salvar/excluir, desabilitar todos os inputs
4. Se `casoId !== 'new'`: buscar dados do caso em Supabase com `useQuery`. Se RLS bloquear (caso de outra subseção sem permissão), exibir "Acesso negado" e navegar de volta.
5. Se `casoId === 'new'`: inicializar com objeto vazio. `subsection_id` = `subsectionId` do auth context (não editável pelo usuario).

**Layout do formulário — reusar estilo do CaseDetailsEditor existente:**
- Header sticky com: botão Voltar (navega para dashboard do role), título, botões Salvar + Excluir
- Card 1 "Informações do Caso": nome (required), processo (required), comarca, UF, vara, especialista, passivo, credores (number), link_habilitacoes (url)
- Card 2 "Datas": ajuizamento (date input), deferimento (date input)
- Card 3 "Documentos" com Accordion igual ao CaseDetailsEditor:
  - Seções: Peças Principais, Relatórios de Atividades, Atas de Assembleias, Incidentes, Ofícios e Editais, Listas de Credores
  - Cada seção usa DocumentUploadEditor com `siteId={OAB_MA_SITE_ID}` e `bucket="site-assets"`

**handleSave:**
```typescript
const handleSave = async () => {
  if (!casoData.nome || !casoData.processo) {
    toast.error("Preencha os campos obrigatórios: Nome e Processo");
    return;
  }

  setSaving(true);
  try {
    if (casoId === 'new') {
      const { error } = await supabase.from('casos').insert({
        subsection_id: subsectionId,  // do AuthContext
        nome: casoData.nome,
        processo: casoData.processo,
        // ... demais campos
        documentos: casoData.documentos || {},
      });
      if (error) throw error;
      toast.success("Caso criado com sucesso!");
      navigate('/portal/presidente');
    } else {
      const { error } = await supabase.from('casos')
        .update({ ...casoData, updated_at: new Date().toISOString() })
        .eq('id', casoId);
      if (error) throw error;
      toast.success("Caso salvo com sucesso!");
    }
  } catch (err: any) {
    toast.error(err.message || "Erro ao salvar caso");
  } finally {
    setSaving(false);
  }
};
```

**handleDelete:**
```typescript
const handleDelete = async () => {
  if (!confirm('Tem certeza que deseja excluir este caso permanentemente?')) return;
  setSaving(true);
  try {
    const { error } = await supabase.from('casos').delete().eq('id', casoId);
    if (error) throw error;
    toast.success("Caso excluído.");
    navigate('/portal/presidente');
  } catch (err: any) {
    toast.error(err.message || "Erro ao excluir caso");
  } finally {
    setSaving(false);
  }
};
```
